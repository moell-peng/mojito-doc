# 简介

Mojito Admin 是基于 Laravel、Vue、Element 构建的基础后台管理系统。

## 截图

![](http://ww1.sinaimg.cn/large/7a679ca1gy1fvrktohzfaj213x0ieq3j.jpg)

## 特征

* 可快速衍生多个后台系统
* 内置角色，权限，用户，菜单管理
* OAuth 2.0，并支持多表鉴权
* 完善的PHPUnit测试
* API 权限精确至路由，页面权限精取到按钮或链接
* 前后端分离
* 多标签页
* 前端支持多语言配置
* 简洁的布局

## 要求

- Laravel  >= 5.5.0
- Vue >= 2.5.17
- Element >= 2.4.6
